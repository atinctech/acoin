ERC20

ERC20 tokens are blockchain-based assets that have value and can be sent and received. It is similar to Bitcoin with few differences. Instead of running on their own blockchain, ERC-20 tokens are issued on the Ethereum network.

Safemath

ATINC COIN is developed by using Solidity Programming Language. Our contract is using the SafeMath library to prevent overflows so it can protect our Smart Contract when it is dealing with number operations.

Smart Contracts

A smart contract is a self-executing contract with the terms of the agreement between buyer and seller being directly written into lines of code. The code and the agreements can be found on the decentralized blockchain network.
